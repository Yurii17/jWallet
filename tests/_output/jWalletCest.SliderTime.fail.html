Двухэтапная проверка

Двухэтапная проверка — это мера безопасности, призванная предотвратить несанкционированный доступ к вашей учетной записи, даже если злоумышленнику известен пароль.

Есть несколько вариантов на выбор::

  • ПИН-Код - дополнительный пароль, с помощью которого Вы будете подтверждать каждый платеж с jWallet.
  • Google Authenticator - "брелок", приложение к телефону, с помощью которого Вы будете подтверждать каждый платеж с jWallet.
